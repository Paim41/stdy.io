# stdy.io

A responsive, front-end learning platform demo built with HTML, CSS, and vanilla JavaScript. Course data, enrollment, progress, cart, checkout, forum activity, and user sessions are simulated in browser storage.

## Highlights

- Emerald light and dark themes using the supplied stdy.io artwork
- Twelve realistic courses with searchable catalog and detailed curricula
- Whole-card navigation and responsive course detail layouts
- One-click learner demo with free-course enrollment confirmation
- Paid-course cart and simulated checkout with safe demo autofill
- Learning progress, quizzes, certificates, dashboards, and community forum
- Responsive layouts for desktop, tablet, and mobile

## Run locally

No build step is required. Open `index.html` directly, or serve this folder with any static file server.

This project has no backend. Authentication, payment, email, and persistence are simulated locally in the browser. Do not enter real personal or financial information.

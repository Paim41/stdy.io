# Image sources

## Illustrated profile avatars

Profile illustrations use the [DiceBear Adventurer](https://www.dicebear.com/styles/adventurer/) avatar style, a remix of Adventurer by Lisa Wischofsky licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). The SVG files are stored locally so the demo does not rely on the avatar API at runtime.
